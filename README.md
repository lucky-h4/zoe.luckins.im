# zoe.luckins.im

Personal professional profile for Zoe Luckins, Senior Ward Sister at Manx Care.

Hosted via GitHub Pages at [zoe.luckins.im](https://zoe.luckins.im).

## Structure

```
/
├── index.html        — single-page site
├── assets/
│   └── style.css     — all styles
├── CNAME             — custom domain
└── README.md
```

## Setup

1. Push to the `main` branch of `https://github.com/lucky-h4/zoe.luckins.im`
2. Enable GitHub Pages in repo Settings → Pages → Source: `main`, folder: `/ (root)`
3. Add a CNAME DNS record: `zoe.luckins.im → lucky-h4.github.io`

## Updating

- Edit `index.html` to update copy or sections
- Edit `assets/style.css` for visual changes
- LinkedIn URL in index.html is a placeholder — update `href="https://www.linkedin.com/in/"` with Zoe's profile URL
